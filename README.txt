Add and use repeater field in Advanced Custom Fields (ACF) free version.
Use any ACF field as repeater field including fields from other plugins as well!

== Installation & Usage ==

1. Install the plugin
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Now you should see repeater field enabled in ACF field group editor.
